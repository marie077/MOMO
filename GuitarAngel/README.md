A Guitar Open String Simulator

Controls

- 'a' key = E low string
- 's' key = A string
- 'd' key = D string
- 'f' key = G string
- 'j' key = B string
- 'k' key = E high string


- Press 'q' to turn off guitar notes
- Press 'w' to turn on guitar notes again

Perfect for tuning, messing around with open strings and playing simple guitar hero :))
